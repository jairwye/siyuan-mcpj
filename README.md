# siyuan-mcpj

思源笔记 MCP 服务器：基于 [MCP](https://modelcontextprotocol.io/) 将思源内核 API 封装为约 49 个工具，支持本机或 Docker 运行，可与 mcpo、Open WebUI、Cursor 等对接，支撑知识库问答与 AI 总结写回。

- **快速开始**：克隆后进入 [siyuan-mcp](siyuan-mcp/) 目录，或使用 Docker 镜像 `ghcr.io/jairwye/siyuan-mcpj:latest`。
- **完整文档**（环境变量、mcpo/Cursor 配置、Docker Compose 等）：见 **[siyuan-mcp/README.md](siyuan-mcp/README.md)**。
